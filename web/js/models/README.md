# 3D Models

Place your 3D model files (.gltf, .glb, .obj, etc.) in this folder.

## Supported Formats:
- GLTF (.gltf)
- GLB (.glb) 
- OBJ (.obj)
- FBX (.fbx)

## Example Usage:
- `models/building.gltf`
- `models/car.glb`
- `models/tree.obj`

The models will be loaded by the Three.js layer module and can be positioned on the map using geographic coordinates.
